# 《小蝌蚪找妈妈》V2 — 技术方案（务实版）

## 目标
达到视效优秀毕业生水平的水墨动画短片。

## 硬约束
- LLM生成代码，不能训练模型
- 单HTML文件，浏览器渲染
- 1280x720 30fps 90秒，Playwright逐帧截图+FFmpeg合成
- 必须能分步迭代，每步可验证

## V1为什么差
用ellipse画圆形当角色，lineTo画直线当腿。和"水墨画"没有关系。
后处理贴上去也没用——地基是歪的。

## V2核心策略

**不追求物理精确模拟毛笔，而是用Canvas 2D API组合出"看起来像水墨"的视觉效果。**

```
关键认知：观众看到的是结果，不是过程。
只要结果看起来像墨在宣纸上晕开的效果就行。
```

## 技术方案

### 一、视觉基础层：笔墨感（如何画出"水墨味"）

V1的问题是用ellipse画实心圆。V2的画法：

**轮廓线：不是一条线，而是多条半透明线叠加**
```js
// 画一个"有笔触感"的形状
function inkStroke(ctx, points, baseWidth) {
  // 1. 主路径 — 粗线、低透明度
  ctx.lineWidth = baseWidth;
  ctx.strokeStyle = 'rgba(20,20,20,0.7)';
  ctx.stroke(path(points));

  // 2. 偏移路径x3 — 细线、极低透明度，模拟笔毛散开
  for (let i = 0; i < 3; i++) {
    const offset = (Math.random() - 0.5) * baseWidth * 0.3;
    ctx.lineWidth = baseWidth * (0.3 + Math.random() * 0.4);
    ctx.strokeStyle = `rgba(20,20,20,${0.1 + Math.random() * 0.15})`;
    ctx.stroke(path(points.map(p => ({
      x: p.x + offset + noise(p.x, p.y, frame),
      y: p.y + offset + noise(p.x, p.y, frame)
    }))));
  }
}
```

**填充：不是纯色，而是径向渐变+噪声遮罩**
```js
// 墨色填充 — 中心浓、边缘淡、有渗透感
function inkFill(ctx, shape, color) {
  // 先用低透明度填充大一点的范围（模拟墨迹渗透）
  ctx.fillStyle = withAlpha(color, 0.08);
  ctx.fill(expandShape(shape, 3));
  // 再用正常透明度填充
  ctx.fillStyle = withAlpha(color, 0.85);
  ctx.fill(shape);
}
```

**宣纸底纹：预生成一次，每帧复用**
```js
// 程序化宣纸纹理 — 噪声函数生成纤维感
function generatePaperTexture(w, h) {
  const canvas = offscreen(w, h);
  const ctx = canvas.getContext('2d');
  // 底色：米白
  ctx.fillStyle = '#f5f0e6';
  ctx.fillRect(0, 0, w, h);
  // 纤维噪声：很多随机细线
  for (let i = 0; i < 2000; i++) {
    ctx.strokeStyle = `rgba(180,170,155,${Math.random() * 0.1})`;
    ctx.beginPath();
    ctx.moveTo(Math.random()*w, Math.random()*h);
    ctx.lineTo(Math.random()*w, Math.random()*h);
    ctx.lineWidth = Math.random() * 0.5;
    ctx.stroke();
  }
  return canvas;
}
```

### 二、角色系统：用贝塞尔曲线定义形态

**蝌蚪（主角）**
```
头部：椭圆轮廓 + inkStroke处理
眼睛：墨点 + 白色高光（很小的圆）
尾巴：贝塞尔曲线，每帧用sin扰动控制点
    → 尾巴根粗尖细，用变宽度stroke
    → 运动时有飘逸感（2-3个控制点各自sin偏移）
身体：不画！蝌蚪头部和尾巴就够了，中间是自然过渡
```

**青蛙妈妈**
```
身体：大椭圆 + inkStroke
眼睛：两个突出的圆（经典青蛙造型）
嘴巴：一条微笑弧线
前腿/后腿：各2条贝塞尔曲线
肚皮：浅色填充（淡墨效果）
```

**配角（鲤鱼/螃蟹/乌龟）**
```
各自用贝塞尔曲线+inkStroke画轮廓
用不同墨色浓淡区分远近
用大小+位置暗示距离
```

**蝌蚪→青蛙变态发育**
```
第0-30s: 纯蝌蚪形态
第30-60s: 尾巴逐渐缩短，后腿长出（曲线从无到有）
第60-75s: 前腿长出，尾巴消失，体型变圆
第75-90s: 完全青蛙形态
所有过渡用参数插值，不是切换精灵图
```

### 三、水下氛围

**水面（画面顶部）**
```
正弦波叠加（3-4个不同频率/振幅）
波浪下方填充半透明蓝绿色
波浪线上有白色高光线
水面有缓慢的颜色脉动
```

**焦散光纹（投射到水底的光斑）**
```
多个半透明椭圆，缓慢移动+旋转
用低透明度叠加模式(screen/lighten)
不需要shader，Canvas2D就够
```

**水下雾气**
```
Canvas2D gradient：从上到下
  顶部透明 → 底部半透明蓝绿
  随时间微妙变化
```

**气泡**
```
上浮粒子，轻微水平摆动
大小2-8px，有高光点
透明度随机
```

### 四、场景元素

**水草**
```
贝塞尔曲线，根部固定，尖端自由摆动
sin波驱动 + 多段不同相位
inkStroke渲染
```

**荷叶**
```
贝塞尔曲线勾勒的圆弧（带缺口）
叶脉：放射线
水珠：小高光圆点
有轻微上下浮动
```

**卵泡（开场景）**
```
半透明椭圆+内部小黑点
破裂时分裂动画
```

### 五、运动与动画

**不用骨骼系统，用弹簧物理**
```
角色移动：目标点 + 弹簧阻尼（自然减速）
尾巴摆动：sin + 噪声（每帧微扰）
水草：多段弹簧链
所有运动都用smoothstep/easeInOutCubic缓动
```

**squash & stretch**
```
角色游动时水平拉伸（stretch）
碰到东西/转向时垂直压扁（squash）
幅度很小（5-15%），但观感明显
```

### 六、电影系统

**虚拟摄影机**
```
translate/scale控制画面偏移和缩放
实现推镜头（scale渐大）、摇镜头（translate渐变）
加轻微噪声模拟手持感
```

**转场**
```
黑场淡入淡出（最稳妥）
水墨化开：可以用Canvas2D filter:blur() + alpha渐变模拟
```

**字幕**
```
保留V1的字幕系统，优化字体和位置
```

### 七、色调与后处理

**整体色调**
```
暖色overlay（非常淡的黄色，alpha 0.03-0.05）
画面有微妙的暖调，不是冷冰冰的纯色
```

**暗角**
```
径向渐变，边缘压暗30-40%
```

**胶片颗粒**
```
预生成噪声纹理，每帧随机偏移
用overlay混合模式叠加
```

**全部用Canvas2D实现，不上WebGL。**

## 渲染管线（每帧）

```
1. 绘制宣纸底纹（预生成，drawImage一次）
2. 绘制远层（焦散光纹）
3. 绘制中层（水草、荷叶）
4. 绘制角色（配角→主角→青蛙妈妈）
5. 绘制近层（气泡、前景水草）
6. 绘制水面波纹（顶部）
7. 绘制水下雾气（gradient overlay）
8. 应用后处理（色调+暗角+颗粒）
9. 输出
```

全部Canvas2D，无WebGL。单线程，无帧缓冲。

## 与V1方案的关键差异

| | V1 | V2 |
|---|---|---|
| 角色画法 | ellipse实心圆+fillStyle | 贝塞尔曲线+inkStroke多层叠加 |
| 水墨感 | 无（只是用了深色） | 宣纸纹理+墨迹渗透+笔触飞白 |
| 水下光影 | 无 | 焦散+雾气+水面波纹 |
| 动画 | 线性插值+sin摆动 | 弹簧物理+噪声扰动+缓动曲线 |
| 后处理 | bloom+暗角+颗粒 | 色调+暗角+颗粒（去掉bloom，水墨画不需要发光） |
| 渲染 | Canvas2D | Canvas2D（纯Canvas，不引入WebGL） |
| 代码量 | ~1900行 | 预计2000-2500行 |
| 调试难度 | 低 | 低（全部Canvas2D，所见即所得） |
